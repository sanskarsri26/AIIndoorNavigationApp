
# AI Indoor Navigation System
**MVP** - AI-powered indoor navigation system with AR capabilities for shoppers, retailers, and brands. Features real-time navigation, product search, inventory management, and analytics dashboards. Built with React, TypeScript, and Capacitor for iOS native support. This is a minimum viable product demonstrating core functionality for smart retail navigation.

## Features

✅ **AR Navigation** - Camera-based indoor navigation with distance and direction indicators  
✅ **Beacon System** - Virtual beacons for precise positioning  
✅ **AR Promotions** - Nearby aisle promotional overlays  
✅ **iOS Native App** - Full native iOS support via Capacitor  
✅ **Retailer Dashboard** - Analytics, heatmaps, inventory management  
✅ **CPG Dashboard** - Brand analytics and targeting  
✅ **ADA-Friendly Routing** - Accessibility features  
✅ **Ghost Inventory Detection** - Advanced inventory tracking  

## Quick Start

### Web Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev
```

### iOS Native App (iPhone 17 Pro)

```bash
# Run automated setup script
./setup-ios.sh

# Or manually:
npm install
npm install @capacitor/core @capacitor/ios @capacitor/camera @capacitor/device @capacitor/splash-screen
npm run build
npx cap add ios
npx cap sync ios
npx cap open ios
```

Then in Xcode:
1. Configure signing & capabilities
2. Select iPhone 17 Pro as target
3. Build and run (Cmd + R)

**See [IOS_SETUP.md](./IOS_SETUP.md) for detailed iOS setup instructions.**

## Build Commands

```bash
# Build for web
npm run build

# Build and sync to iOS
npm run ios:build

# Sync web assets to iOS
npx cap sync ios

# Open in Xcode
npx cap open ios
```

## Project Structure

- `src/` - React source code
- `ios/` - iOS native project (generated by Capacitor)
- `capacitor.config.ts` - Capacitor configuration
- `vite.config.ts` - Vite build configuration (optimized for fast builds)

## iOS Requirements

- macOS with Xcode 15.0+
- iOS 15.0+ (supports iPhone 17 Pro)
- Apple Developer Account (for device deployment)
- CocoaPods installed
